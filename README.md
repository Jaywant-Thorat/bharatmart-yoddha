# BharatMart Yoddha — DP-600 Fabric Solution
 
End-to-end Microsoft Fabric analytics solution for **BharatMart**
(pan-Bharat retailer, 43 stores, 4 zones), built during the
**Fabric Yoddha Bootcamp** (#MissionPowerBIBharat).
 
**Author:** Jaywant Thorat  ·  **Goal:** DP-600 certification
 
## Structure
- `data/raw/`   - the 5 BharatMart source CSVs (50,569 rows)
- `sql/`        - SQL endpoint & Warehouse scripts (Week 4)
- `notebooks/`  - PySpark Silver notebooks (Week 3)
- `pbip/`       - Power BI project files (Weeks 5-6)
- `docs/`       - ERD, screenshots, notes
 
## Validation anchor
Total revenue = **Rs.84.3 Cr** across 5 fiscal years - checked at
every layer (Bronze, Silver, Gold, model, Prod).
